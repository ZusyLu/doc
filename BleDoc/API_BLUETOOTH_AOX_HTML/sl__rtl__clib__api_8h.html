This is the API for the Silicon Labs Real-Time Locationing library. It provides an interface for estimating arrival and departure angles of signals and the positions of AoA/AoD signal transmitters. AoA stands for Angle-of-Arrival and AoD for Angle-of-Departure. AoX is used when referring to both techniques.

Estimators can be created individually for each locator node using the API. Even multiple estimators with different parameters can be created for a single node. An instance of the estimator is created as a libitem and the estimator is initialized. Next, the estimation parameters such as antenna array type, number of antennas, estimation mode, and so on can be set using the function calls described below.

One snapshot means a set of IQ-samples with exactly one sample per each antenna.
