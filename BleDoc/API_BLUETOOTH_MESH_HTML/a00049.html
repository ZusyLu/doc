Data Fields
uint8_t reason

Reason for failure. Values are as follows:

  • 0: Link closed
  • 1: Invalid PDU
  • 2: Invalid PDU format
  • 3: Unexpected PDU
  • 4: Confirmation failed
  • 5: Out of resources
  • 6: Decryption failed
  • 7: Unexpected error
  • 8: Unable to assign address
