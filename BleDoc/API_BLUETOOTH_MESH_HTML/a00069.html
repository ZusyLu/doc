sl_status_t sl_btmesh_health_client_get ( uint16_t  server_address,
uint16_t  elem_index,
uint16_t  appkey_index,
uint16_t  vendor_id 
)

Get the registered fault status of a Health Server model or models in the network.

Besides the immediate result code, the response or responses (if the destination server address is a group address) from the network will generate sl_btmesh_evt_health_client_server_status.

Parameters
[in]server_addressDestination server model address. May be a unicast address or a group address.
[in]elem_indexClient model element index. Identifies the client model used for sending the request.
[in]appkey_indexThe application key index to use in encrypting the request
[in]vendor_idBluetooth vendor ID used in the request
Returns
SL_STATUS_OK if successful. Error code otherwise.
