Detailed Description

Receiving a Health Server fault status message generates this event.

The Client model may receive a status message because:

  • * it made a sl_btmesh_health_client_get to which a Server model responded, or
  • * it made a sl_btmesh_health_client_clear to which a Server model responded, or
  • * it made a sl_btmesh_health_client_test to which a Server model responded.
