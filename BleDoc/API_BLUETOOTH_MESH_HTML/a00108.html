Detailed Description

This event contains a list of key indices for the application keys which are bound to a model.

The list is requested using the sl_btmesh_config_client_list_bindings command. More than one such event may be generated; the list contents are terminated by a sl_btmesh_evt_config_client_bindings_list_end event.
